Como usar

  1. Toque no botão do sinal desejado para iniciar o áudio.
  2. Acompanhe o progresso na barra inferior e clique nela para avançar/voltar.
  3. Ajuste o volume do dispositivo para melhor experiência.
  4. Recomenda-se o uso de fones em ambientes de estudo.
